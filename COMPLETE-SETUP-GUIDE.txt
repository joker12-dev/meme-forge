Problem: npm install fails
Solution: npm install --legacy-peer-deps --verbose

Problem: Database connection refused
Solution: 
  - Check PostgreSQL: sudo systemctl status postgresql
  - Check DATABASE_URL in .env
  - Test: psql -U memeforge -d memeforgedb -h localhost

Problem: NGINX shows 502 Bad Gateway
Solution:
  - Check backend running: pm2 status
  - Check NGINX config: sudo nginx -t
  - View NGINX logs: sudo tail -f /var/log/nginx/error.log

Problem: SSL certificate error
Solution:
  - Verify DNS is pointing to server IP
  - Manually renew: sudo certbot renew --force-renewal
  - Check cert: sudo certbot certificates

Problem: Port already in use
Solution:
  - Find process: lsof -i :3001
  - Kill: kill -9 <PID>
  - Or restart PM2: pm2 restart all

Problem: PM2 process keeps crashing
Solution:
  - View logs: pm2 logs
  - Check .env file exists
  - Check file permissions
  - Restart: pm2 restart all
